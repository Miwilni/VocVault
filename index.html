Beschreibung

Willkommen bei VocVault!
VocVault ist ein momentan noch kostenloser moderner Vokabeltrainer, mit dem Sie Sprachen und Vokabeln effektiv und smart lernen können.

Bereits implementierte Funktionen

  • Lernmodus
  • Konsolenprogramm
  • Alle Vokabeln oder nur eine bestimmte Anzahl lernen
  • Vokabeln zufällig Abfragen
  • Ausführliche Abfragestatistik über die derzeitige Abfrage
  • Automatische Spracherkennung der angegebenen Vokabeln
  • Möglichkeit der Variation, von welcher in welche Sprache die Vokabeln abgefragt werden sollen
  • Noch ein paar kleine weitere Funktionalitäten
  • Vokabeleinlese:
    • aus csv Datei
    • im csv Format (mit Trennzeichen getrennt)
    • im Python Dictionary Format
  • Momentan in Bearbeitung:
    • UI Programmierung
    • Cloud-Synchronisation zwischen Geräten und Benutzerverwaltung über eine MySQL Datenbank

Geplante Features

  • Ausführliche Nutzungs- und Abfragestatistik
  • Uploadmöglichkeit in der UI um csv/ Jason Dateien einzulesen
  • Automatische Übersetzung eingegebener Vokabeln bzw. überprüfung der Richtigkeit mit dem Wörterbuch
  • Kombination von Benutzerverwaltung und Vokabeltrainer zur Individualisierung
  • Import/Export von Vokabellisten
  • Mehrsprachige Benutzeroberfläche
  • Mobile App-Version für Android/iOS
  • Quiz- und Spielmodi zum spielerischen Lernen
  • Benutzerfreundliches Interface für schnelle Eingabe
  • Automatische Erstellung von VocabVaults zu bestimmten Themen zum freien Lernen von Vokabeln einer Sprache

Download

Momentan steht VocVault leider noch nicht als Applikation zur Verfügung, da die UI Entwicklung noch nicht abgeschlossen ist, ich bitte um Verständnis!

Dennoch können Sie sich den aktuellen Stand von VocVault auf Github ansehen, sowie eine Python Datei mit einer pre-Version des Vokabeltrainers als Konsolenprogramm ohne Benutzerverwaltung und Cloud Support herunterladen.

Sollten Sie sich die Pre-Version von VocVault herunterladen wollen, ist es notwendig, dass Sie Python installiert haben und die Datei mit Python ausführen über Rechtsklick --> "öffnen mit" und dann Python 3.[Ihre Version]! Python können Sie auch bequem über den Button unten downloaden. Nach dem Download der python[Version u. Betriebssystemsdaten].exe Datei müssen Sie zunächst diese ausführen und das Setup durchklicken, bevor Sie die VocVault.py Datei ausführen können.

Weiterhin sollten bei der Pre-Version die CSV-Dateien in einem Ordner namens "csv_documents" liegen und die Python Datei ebenfalls in einem anderen Ordner mit einem Namen Ihrer Wahl. Diese beiden Ordner sollten im gleichen Oberordner liegen! Diese Anforderung ergibt sich aufgrund der momentanen Projektstruktur für die endgültige Version, danke für ihr Verständnis!
